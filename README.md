# Claude GUI Tool

## 개요
- Cluade를 GUI로 사용하기 위한 도구입니다.
- Cluade의 인증방식은 Claude.ai 계정 인증으로 되어 있음.
- 현재 터미널 방식의 사용을 GUI로 대체하여 사용자 경험을 향상시킵니다.

## 주요사항
- 빌드는 윈도우와 맥 두가지 버전으로 빌드할 수 있어야함.
- 화면에서 5가지 탭을 만들수 있어야 함.
- 탭 내에서 진행 중 또는 정리된 plan의 결과가 보기 좋게 노출될 수 있는 상위 화면이 있고, 하위에 입력을 받을 수 있은 editText가 있어야 함.
- 최상위 탭에서 하위에 4가지 탭을 관리할 수 있는 기능을 체크박스로 옵션으로 설정할 수 있어야함.
- 파일등은 드래그앤 드랍으로 바로 첨부가능.
- 첨부된 이미지도 미리보기로 보여져야 함.
- Claude가 작성한 plan은 마크다운으로 가시성이 좋아함.
- plan 모드는 기본으로 사용할 수 있는 체크박스가 있어야함.
